### 移动端适配

1.  简单一点的页面，一般高度直接设置成固定值，宽度一般撑满整个屏幕。
2.  稍复杂一些的是利用百分比设置元素的大小来进行适配，或者利用 flex 等 css 去设置一些需要定制的宽度。
3.  再复杂一些的响应式页面，需要利用 css3 的 media query 属性来进行适配，大致思路是根据屏幕不同大小，来设置对应的 css 样式。
